# Tail-Command
Tail Command in node.js

Follwoing steps to setup :-
1. git clone https://github.com/shhhRakesh/Tail-Command.git
2. cd Tail-Command
3. sudo npm install -g
4. nail <filename.txt> 
    e.g. 
    for one file :- 
              nail myfile.txt 
    for multiple files :-           
              nail myfile.txt readme.txt
